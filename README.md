# Ultrasonic-sensor
Implement a system to measure the distance using ultrasonic sensor 
HC-SR04 and Atmega32

## Main Components
- ATmega-32 
- 4*16 LCD
- Ultrasonic Sensor
- oscilloscope

## Drivers
- ICU
- LCD
- GPIO
- Ultrasonic
